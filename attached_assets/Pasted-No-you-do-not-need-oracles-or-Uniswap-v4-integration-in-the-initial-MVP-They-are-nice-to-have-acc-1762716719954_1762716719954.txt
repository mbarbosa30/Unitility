MVP Scope: “Gasless Send + Manual Rebalance”
What Ships in v0.1
	1	Pool Factory → Anyone deposits ETH → picks token → sets fee% + minTransfer
	2	Paymaster Pool (ERC-4337) → Validates: amount >= minTransfer, pool has ETH for gas → Executes: transfer(amount - fee) to receiver, fee to sponsor → Deducts gas from pool ETH
	3	Frontend Dashboard
	◦	Send Modal: Pick token → auto-select valid pool (lowest fee% among those with minTransfer <= amount)
	◦	Pool List: Shows fee%, minTransfer, ETH balance, volume
	◦	Sponsor View: Deposit, withdraw, adjust params
	◦	Rebalancer View: Manual “Rebalance” button → Shows: Pool: $DOGGO
	◦	ETH Balance: 4.8
	◦	Fees Accrued: 1,240 $DOGGO
	◦	[Swap Fees → ETH on Uniswap v3] ← one-click
	◦	
No Oracle? No Problem.
	•	Discount calculation: Offchain only → Use The Graph to index pool state + Coingecko API (or 1inch quote) for spot price → Show in UI: “Est. Discount: -8.2%” (with disclaimer)
	•	Rebalancing: Manual swap → Sponsor clicks → dApp routes token → WETH via Uniswap v3 router (no v4 needed) → Profits go back to wallet
This is 100% functional, auditable, and secure.

   → +0.5 $DOGGO in pool
   → ETH balance: 4.799 (after gas)

   → "Fees accrued: 1,240 $DOGGO"

Security Note: No Oracle = Safer MVP
	•	No price manipulation → can’t drain pool via fake oracle
	•	No hook complexity → simpler audit
	•	Fail-safe: If offchain price wrong → user still sends, just no arb

Bottom Line: Ship v0.1 Without Them
Build the send. Let the market teach you the arb.
